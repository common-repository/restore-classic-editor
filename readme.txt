Remove Gutenberg Editor and get back to old version of editor. This provides Original Classic Editor and more.

== Description ==

Have you upgraded WordPress and stucked with the new editor? Don't worry, Just install the Remove Gutenberg plugin and enjoy the Classic editor. Additionally, There is an option in Reading settings from you can enable/disable the feature.

== Frequently Asked Questions ==

= Can I disable Gutenberg feature? =

Yes, We can disable the Gutenberg plugin without deactivating or removing the plugin. Go to Settings > Readings > Remove Gutenberg Settings and check Disable Gutenberg option.

= Do I need to do anything to Remove Gutenberg? =

No, Just Install, activate and Gutenberg will be removed.

= Why should we Remove Gutenberg? =

Many WordPress themes are working on third party ready-made page builders that create conflict with Gutenberg at the moment we can remove Gutenberg and make it page builder works.

== Installation ==

Upload the Remove Gutenberg plugin to your website, Activate it, and enjoy :).

1, 2, 3: You're done!
